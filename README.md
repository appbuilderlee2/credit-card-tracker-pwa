# Credit Card Tracker (PWA)

A lightweight, static PWA dashboard for the **OpenClaw - Credit Cards** Google Sheet.

## What it is (MVP)
- Static web app (works on GitHub Pages)
- Home dashboard + card list UI based on the provided Stitch mock
- Optional data source: Google Sheets (via **Google Visualization API**) if you publish a sheet/tab.

## Data source options
### Option A (no backend): Publish a sheet/tab
1. In Google Sheets: **File → Share → Publish to web**
2. Publish either the whole sheet or a specific tab.
3. Put the published URL/tab name in `src/config.js`.

### Option B (private sheet): requires a backend
If the sheet must stay private, GitHub Pages can’t fetch it directly without OAuth.
In that case we’ll add a small proxy/API (Cloudflare Worker / server) later.

## Dev
This MVP is plain HTML/CSS/JS (no build step).
- Entry: `index.html`
- Code: `src/`

## Deploy (GitHub Pages)
- Uses the `gh-pages` branch generated by GitHub Actions.
